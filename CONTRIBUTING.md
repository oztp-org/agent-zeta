# Contributing to Agent Zeta

Thank you for your interest in Agent Zeta and the Open Zero Trust Project.

## How this project is maintained

Agent Zeta is maintained by a small core team. At this stage of the project,
we are **not accepting external pull requests**. All code changes are reviewed
and committed by the core team.

This is a deliberate security posture. Software supply chain attacks — where
malicious code is introduced through seemingly legitimate contributions — are
one of the fastest-growing threat vectors in open source. For a project built
around Zero Trust principles, practicing what we preach means applying the same
scrutiny to our own codebase.

## What you can do

- **Use the software.** Everything here is free and open source under the
  project license. Fork it, run it, build on it.
- **Open an issue.** Bug reports, documentation gaps, and feature ideas are
  welcome via GitHub Issues. We read all of them.
- **Star or share the project.** The best way to support OZTP right now is
  helping others find it.

## Reporting a security vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report privately by emailing the project maintainer directly. We will
acknowledge within 72 hours and work with you on responsible disclosure.

## Questions

General questions about Zero Trust implementation or how to use OZTP tools
are welcome in GitHub Discussions or via the chat widget at
[oztp.org](https://oztp.org).
